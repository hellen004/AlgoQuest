AlgoQuest: A 2D Puzzle Game for Learning Algorithms
Project Overview
AlgoQuest is an educational 2D game designed to teach university students 10 core algorithms (Bubble Sort, Merge Sort, Linear Search, Binary Search, Dijkstra’s Algorithm, A* Pathfinding, Recursion, Dynamic Programming, BFS, DFS) through interactive puzzles. Built in Godot using GDScript, the game emphasizes visualization, interactivity, and gamification to make learning engaging.
Problem Statement
Learning algorithms is challenging due to their abstract nature and lack of interactive teaching methods. This project targets Zambian university students, providing a 2D game to visualize and practice algorithms, improving retention and engagement.
Project Objectives

Develop a 2D educational game to teach 10 algorithm concepts interactively.
Implement gamified levels for each algorithm (Bubble Sort, Merge Sort, Linear Search, Binary Search, Dijkstra’s Algorithm, A* Pathfinding, Recursion, Dynamic Programming, BFS, DFS).
Create intuitive 2D visuals to demonstrate algorithm behavior.
Ensure a rewarding user experience with scoring and feedback.

Proposed Solution
A 2D puzzle game in Godot where each level represents an algorithm:

Players interact with 2D sprites (e.g., moving objects for sorting, navigating grids for pathfinding).
Visual cues (e.g., highlighted nodes, animated swaps) show algorithm steps.
Bonus challenges test understanding (e.g., predict the next step in Bubble Sort).
Scoring system rewards correct interactions and completion speed.

Why Component-Based?

Modular design allows independent development of algorithm levels.
Reusable components for visuals and interactions.
Scal personally for future additions (e.g., new algorithms).

AlgoQuest will offer an engaging, interactive way to learn algorithms, reducing learning barriers and promoting understanding through 2D gameplay. The project is ready for implementation with clear objectives and a feasible timeline.
